# TaskFlow 📋

Application de gestion de tâches développée en Java.

## 🛠️ Technologies
- Java 21
- POO (Programmation Orientée Objet)
- Collections, Exceptions, Scanner

## 📁 Structure du projet
```
TaskFlow-Practice/
├── src/
│   ├── Main.java
│   ├── Tache.java
│   └── GestionnaireTaches.java
└── docs/
    ├── erd-taskflow.png
    └── database.sql
```

## 🗄️ Base de données
![ERD TaskFlow](docs/erd-taskflow.png)

## ✨ Fonctionnalités
- Ajouter une tâche
- Afficher toutes les tâches
- Terminer une tâche
- Menu interactif en ligne de commande
- Gestion des erreurs de saisie
